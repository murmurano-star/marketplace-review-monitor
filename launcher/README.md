# HTML-запуск мониторинга на Windows

Папка `launcher` содержит локальное HTML-приложение для ручного запуска GitHub Actions.

## Установка ярлыка

1. Скачайте или клонируйте репозиторий на компьютер.
2. Откройте папку `launcher`.
3. Дважды щёлкните `INSTALL_DESKTOP_SHORTCUT.cmd`.
4. На рабочем столе появится ярлык **«Мониторинг отзывов GIGAS и SOFT99»**.

## Первый запуск

Для работы требуются:

- Python 3;
- GitHub CLI `gh`;
- авторизация в GitHub через `gh auth login`.

При первом открытии ярлыка приложение проверит эти компоненты. Если GitHub CLI не авторизован, откроется стандартное окно входа GitHub.

## Работа приложения

Ярлык запускает локальный сервер только на адресе `127.0.0.1:8765` и открывает HTML-интерфейс в браузере. В интерфейсе можно:

- выбрать начало и конец периода;
- быстро установить последние 7 дней;
- запустить workflow `manual-monitor.yml`;
- открыть страницу GitHub Actions.

GitHub-токен не записывается в HTML и не хранится приложением. Авторизация выполняется GitHub CLI.
